# Next Gen Academy

Next Gen Academy is a simple educational website created as a front-end web development project. The aim of this project is to provide students with an easy-to-use platform where they can explore different courses, learn about the academy, and get in touch through the contact section.

The website is built using HTML, CSS, and JavaScript. It has a responsive design, so it works well on desktops, tablets, and mobile devices. This project helped me improve my skills in designing web pages, organizing files, and making websites interactive.

## Features

- Responsive layout
- Home, About, Courses and Contact sections
- Clean and simple user interface
- Navigation menu
- Interactive elements using JavaScript
- Mobile-friendly design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

## About This Project

I developed this project to practice front-end web development concepts such as page layout, responsive design, styling with CSS, and adding basic interactivity with JavaScript. It also helped me understand how to organize a website project properly.
